## Deployment

This site is designed to deploy to Vercel (or Netlify) for free hosting.

### Deploy to Vercel:
1. Go to [vercel.com](https://vercel.com)
2. Sign up or log in
3. Click "Add New" â†’ "Project"
4. Import this GitHub repository
5. Click "Deploy"
6. Update DNS for storvita.com to point to Vercel

### DNS Settings:
Once deployed, Vercel will provide DNS records. Update these at your domain registrar (Squarespace):
- A record pointing to Vercel's IP
- CNAME for www subdomain
- Keep existing MX records for email
